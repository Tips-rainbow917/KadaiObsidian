# 🛠️ コマンド・ワークフロー一覧

このプロジェクトで使用可能なコマンドとワークフローのまとめです。

## 🎤 音声文字起こし関連

| コマンド / ワークフロー | 実行方法 | 内容 |
| :--- | :--- | :--- |
| **音声ワークフロー** | ` /voice` (AIへの依頼) | 「00ボイスメモ」内の音声ファイルを検出し、文字起こし・AI要約・Obsidianノート化・アーカイブまでを自動で行います。 |
| **一括実行スクリプト** | `./.agent/workflows/voice.sh` (ターミナル) | 上記のオートメーションをターミナルから1ステップで実行します。 |
| **メインスクリプト** | `python3 .agent/workflows/process_voice_memos.py` | ワークフローの本体プログラムです。 |

---

## 📁 フォルダ構成と役割

- **`.agent/`**: プロジェクトの全設定・スクリプト・ルールを集約。
  - **`workflows/`**: `/voice` コマンド定義や実行スクリプト（Python, Shell）。
  - **`rules/`**: AIへの指示書やプロジェクト規約。
- **`00ボイスメモ/`**: 入力音声
- **`00ボイスメモ/processed/`**: 処理が終わった音声が自動でここに移動（アーカイブ）。
- **`01文字起こし/`**: AIが整形した Obsidian 用ノート（.md）がここに生成される。

## ⚙️ 設定

- **`.env`**: OpenAI APIキーを管理。
- **`音声文字起こしREADME.md`**: 基本的なツールの説明。
- **`README_ワークフロー.md`**: 自動化ワークフローの詳細な説明。
