# プロジェクトルール

## 基本方針
- すべての応答は日本語で行う。
- 文字起こし結果は `01文字起こし` フォルダに保存する。
- 処理済み音声は `00ボイスメモ/processed` に移動する。
- 複雑な作業の前には必ず `implementation_plan.md` を作成する。

## フォルダ構造の規定
- `.agent/`: エージェント用設定（workflows, rules）
- `.workflows/`: 自動化スクリプト本体
- `.rules/`: プロジェクト固有のルール定義
- `00ボイスメモ/`: 入力音声
- `01文字起こし/`: 書き起こし結果
